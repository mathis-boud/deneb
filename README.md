# 🌌 Deneb Bot

Deneb est un bot Discord polyvalent développé en **Node.js**, conçu pour offrir un système de rappels intelligents et des fonctionnalités communautaires.

## 🛠️ Fonctionnalités
- **Rappels Séquentiels** : Système de relances automatiques à intervalles personnalisables (ex: toutes les 5 min).
- **Persistance des données** : Utilisation de **SQLite** pour garantir qu'aucun rappel n'est perdu, même après un redémarrage.
- **Architecture Pro** : Commandes slash (`/`), boutons interactifs et embeds stylisés.

## 💻 Stack Technique
- **Langage** : Node.js (v22+)
- **Librairie** : Discord.js v14
- **Base de données** : SQLite (via better-sqlite3)
- **Déploiement** : VPS Ubuntu, géré via PM2 et Git.

## ⚙️ Installation (Pour consultation)
1. `git clone https://github.com/mathis-boud/deneb.git`
2. `npm install`
3. Configuration via `config.json`.
